###说明 

IMDmeo 是一个简单的开源即时通信客户端，由MySQL数据库 + Openfire服务器 + Android客户端(使用 Smack 4.19 类库)组成，主要功能：注册账号、登录和退出系统、获取好友列表、添加和删除好友、收发文本消息、分享位置（基于百度地图SDK）。
